LeeKloud 1.0.1
========

Une API pour synchroniser vos IA qui sont sur votre ordinateur avec le site leekwars.com.

Pour l'utiliser installer [nodejs](http://nodejs.org/).

Ensuite lancez cette commande :

    node _LeeKloud.js

Sous windows :

    cd C:\{Chemin_vers_LeeKloud.js}\
    node _LeeKloud.js
